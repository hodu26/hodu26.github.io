## 3. 스터디 개요
국민대 OSTEP 강의 및 자료들을 활용하여,   
운영체제에 관한 기본 지식을 이해도를 증가시키고자 진행

&nbsp;

## 4. 스터디 일정
스터디는 매주 1회씩 진행되며,   
각 주마다 2명씩 주제를 맡아 PPT 준비 및 발표하고, 질문을 통한 이해 검증을 시행 

&nbsp;

## 5. 학습 내용
인트로  ->  CPU 및 프로세스 가상화  ->  프로세스 API  ->  Direct Execution     
->  Limited Direct Execution  ->  CPU 스케줄링  ->  Address Space  ->  ...
